Content Repurposer

快速开始

Content Repurposer API 允许您将内容一键转换为适合各种社交媒体平台的格式。

购买后，您将收到API Key。使用此Key进行身份验证。

curl -H "Authorization: Bearer YOUR_API_KEY" \ -H "Content-Type: application/json" \ -d '{"content": "您的内容", "target_platform": "twitter"}' \ https://content-repurposer-production-34d6.up.railway.app/api/repurpose

API端点

POST /api/repurpose

将内容转换为目标平台格式。

请求参数

参数类型必填说明
contentstring是要转换的原始内容
target_platformstring否目标平台（见下方列表）

支持的平台

twitterlinkedinfacebookinstagram youtubeblogemail

示例响应

{ "success": true, "content": "转换后的内容...", "input_length": 150, "output_length": 200, "plan": "pro" }
GET /api/usage

查询当前账户的使用统计。

示例响应

{ "success": true, "plan": "pro", "total_calls": 42, "max_calls": 500, "total_input_chars": 15000, "total_output_chars": 22000 }
GET /health

健康检查端点，用于验证服务是否正常运行。

错误处理

状态码错误类型说明

400Bad Request请求参数错误或内容过长
429Too Many Requests超出API调用限制
500Internal Server Error服务器内部错误
